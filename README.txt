Random Number Generator

Description:

This web page generates a random array of integers and displays it along with arrays of numbers filtered based on divisibility criteria (divisible by 2, 5, or both). It provides a button to regenerate a new random array.

Files:

- Index.html: HTML file containing the random number generator interface.
- style.css: CSS file for styling the random number generator interface.

How to Use:

1. Open index.html in a web browser.
2. The page will display a random array of integers and arrays of numbers filtered based on divisibility criteria.
3. Click the "Regenerate a New Random Array" button to generate a new random array and update the displayed arrays accordingly.

Code Structure:

- HTML: Defines the structure of the webpage and elements to display random numbers and filtered arrays.
- JavaScript: Contains functions to generate random numbers, filter arrays, and update the displayed arrays.
- CSS: Styles the appearance of the HTML elements.
